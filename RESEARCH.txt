MIFare classic 1k:
- card has 16 sectors with 4 blocks each
- each block has 1024 bytes of storage space
- 1st sectod is read only
- each sector is secured by two 48-bit keys: A and B
- last block in sector contains these keys, as well as a configuration
  that defines the purpose of each key (one is for read-only access,
  and second may be for write access, it defines that for each block)
- empty card has default keys set to FFFFFFFFFFFF or 000000000000
